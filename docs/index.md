# Introduction

!!! abstract "This site contains basic notes for users of the KMN Data Center."

ATU Letterkenny has operated a teaching data center since c. 2010, servicing undergraduate and post-graduate programmes. Research projects are also carried out here.

This site is intended to provide general information for students to be able to access the data center and to understand what the rules are.

No data is to be stored in this data center which is subject to GDPR. This is a teaching data center only.

No SLA is in place for any system, this is an academic facility, all services are best effort only.
