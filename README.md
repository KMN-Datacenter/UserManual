# User Manual
This site was created using Material for MKDocs

1. Create a venv and activate

````
py -m venv venv
.\venv\Scripts\activate.ps1 
````

2. Check the PIP version, upgrade if necessary
````
pip --version
python.exe -m pip install --upgrade pip
````
3. Now install MKDocs Material
````
pip install mkdocs-material
````
4. Create a new site by typing 
````
mkdocs new .
````
